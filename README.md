## Usage

1. **Clone** the repository:
   ```bash
   git clone https://github.com/dantol29/webserver.git
   cd webserver
   ```
2. **Compile** the webserver:
   ```bash
   make
   ```
3. **Run** the server:
   ```bash
   ./webserver
   ```
4. **Visit** `http://localhost:8080` or `http://127.0.0.1:8080` in your browser to view the server response.
